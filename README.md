# SkillBadge BF — Prototype Phase 2

## 📋 Description

SkillBadge BF est une plateforme innovante de certification des compétences basée sur la blockchain, spécialement conçue pour le Burkina Faso. Cette application web permet aux formateurs d'émettre des badges NFT de compétences, aux apprenants de gérer leur portfolio numérique, et aux recruteurs de vérifier l'authenticité des certifications en temps réel.

## ✨ Fonctionnalités Principales

### 🎓 Interface Formateur
- **Tableau de bord statistiques** : Suivi des badges émis, apprenants certifiés et badges actifs
- **Création de badges** : Formulaire complet pour émettre de nouveaux badges NFT
- **Prévisualisation NFT** : Aperçu du badge avant émission sur la blockchain
- **Historique des émissions** : Liste des badges récemment créés avec liens de transaction

### 👤 Interface Apprenant  
- **Portfolio personnel** : Profil complet avec score de confiance dynamique
- **Visualisation des compétences** : Grille interactive des domaines maîtrisés avec barres de progression
- **Collection de badges** : Liste détaillée des certifications obtenues
- **Partage sécurisé** : Génération de liens de vérification avec QR Code
- **Export PDF** : Téléchargement du portfolio en format PDF

### 🔍 Interface Recruteur
- **Vérification blockchain** : Validation en temps réel de l'authenticité des badges
- **Recherche avancée** : Recherche par adresse wallet ou URL portfolio
- **Rapport détaillé** : Visualisation complète du profil candidat vérifié
- **Export et contact** : Téléchargement de rapports PDF et contact direct

## 🛠 Technologies Utilisées

- **Frontend** : HTML5, CSS3, JavaScript (ES6+)
- **Design** : CSS Grid, Flexbox, animations CSS personnalisées
- **Typographie** : Google Fonts (Syne, DM Sans, JetBrains Mono)
- **Blockchain** : Simulation d'intégration ERC-721 (Sepolia Testnet)
- **Responsive** : Design adaptatif mobile-first
